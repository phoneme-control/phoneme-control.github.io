# Phoneme Control Project Site

Static project page for a paper on phoneme-level control in speech generation.

## GitHub Pages Setup

1. Create a GitHub repository named `phoneme-control.github.io` under the
   `phoneme-control` account.
2. Push these files to the repository's default branch.
3. In GitHub, open **Settings > Pages** and publish from the default branch root
   if GitHub does not enable it automatically.
4. The site should become available at
   `https://phoneme-control.github.io`.

## Editing Content

- Update the paper title, author list, abstract, and citation in `index.html`.
- Replace placeholder buttons with the final paper and code links.
- Add audio demos or figures under an `assets/` directory and reference them
  from the Results section.

No build step is required.
